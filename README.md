# 프로젝트 LAP

## 이 레포지토리는 기록용입니다.
> 프로젝트 방향성이 변경되어 다른 저장소에서 개발을 진행합니다.

## Getting Started

### Install Dependencies

- pnpm으로 설치를 권장합니다.

```bash
npm install --legacy-peer-deps
```

### Start the Development Server

```bash
npm run dev


pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.



## 백앤드 서버 실행

```bash
cd py_server


python fast_temp.py
```
