# 프로젝트 LAP



## Getting Started

### Install Dependencies

- pnpm으로 설치를 권장합니다.

```bash
npm install --legacy-peer-deps
```

### Start the Development Server

```bash
npm run dev


pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.



## 백앤드 서버 실행

```bash
cd py_server


python fast_temp.py
```
